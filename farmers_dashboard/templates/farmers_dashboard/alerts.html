How to keep alerts understandable

One action per card

Every alert should point to the next useful screen, not leave the farmer guessing what to do next.

Separate urgent from routine

Critical cases should feel visually different from routine reminders or low-pressure follow-up.

Connect support and AI clearly

Some alerts need a support article, while others need an AI question or a report review.

Keep timestamps visible later

When live alert data is added, include timestamps and status changes so follow-up stays traceable.
